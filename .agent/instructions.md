# Agent Directives: La Ruta a Trascendencia

Estas directivas aseguran que cualquier agente de IA que trabaje en este repositorio mantenga la visión artística y técnica original del proyecto.

## 1. Estética Visual: "Deep Space & Glassmorphism"

El sitio debe evocar la atmósfera de una novela de ciencia ficción dura (Hard SF).
- **Esquema de Colores**: Fondo oscuro profundo (#0a0a0a o similar), acentos neón (Cyan #00f2ff, Violeta #7000ff).
- **Estilo de UI**: Glassmorphism (paneles translúcidos con desenfoque de fondo y bordes sutiles). Aplicar `backdrop-filter: blur(10px)` siempre que sea posible.
- **Tipografía**: Usar fuentes sans-serif modernas y tecnológicas (ej. Google Fonts: "Inter" para lectura, "Michroma" o "Orbitron" para titulares).

## 2. Pautas de Implementación

- **Frameworks**: Priorizar HTML, CSS y JS "vanilla" para máxima portabilidad y control, a menos que el usuario pida explícitamente un framework como Vite o Next.js.
- **Interactividad**: Implementar micro-animaciones suaves (hover effects, transiciones de suavizado) para que el sitio se sienta "vivo".
- **Infografía**: La imagen `Infografia.png` es el corazón visual. Debe ser tratada con respeto técnico (zoom, hotspots, optimización de carga).

## 3. Contexto del Proyecto

- **Obra**: "La Ruta a Trascendencia" de Alejandro Alonso.
- **Temas**: Percepción del tiempo, métrica de Alcubierre, mutación humana, poshumanismo en Latinoamérica.
- **Tono**: Profesional, inmersivo, tecnológico y misterioso.

---

> [!IMPORTANT]
> **No usar placeholders genéricos**. Si se requiere una imagen nueva, usar la herramienta `generate_image` para crear un diseño que combine con la estética Deep Space.

> [!TIP]
